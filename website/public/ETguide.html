Everything you need to play RTCW Enemy Territory free

RTCW:Enemy territory is an epic 13 year old FREE-TO-PLAY online FPS game still played today by thousands of people arround the world. It's glory days are over but if you are looking for a good FPS game with decent graphics and amazing teamwork based gameplay you should definitely give it a shot, its completly free.

To start playing it today all u need to do is follow the simple steps and download the components below.

Check out the video if u are having trouble with this guide and feel free to comment on it if u need more help.

Essentials:

RTCW:Enemy Territory.

Etpro 3.2.6.

Config file.

ET server List.


Optionals:

jaymod.

etpub.

punkbuster.

IRC for scrims.

Trickjump mod.

Step 1:

Download and install RTCW: Enemy territory


Step 2:

Download Etpro and unzip it inside ET install destination


Step 3:

Get the cfg file from Dropbox and place it inside the etpro folder, you must rename the cfg file to autoxec.cfg otherwise u will have to execute it manually everytime u connect to a server.


Step 4:

If you followed the steps ET should be almost ready to play, simply click on et server list, find a etpro public server to play from etsplatter, click on it and then copy the ip adress, open the game, click the \ key on your keyboard to open the console and write /connect (ipadress)


Step 5:

If you didnt rename the cfg file simply open the console (\) and write /exec WiP.cfg in the console, voila! enjoy the game.
